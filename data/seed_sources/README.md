# Seed sources

This directory stores manually curated source lists for reproducible demos.

For the first CLTV demo, fill `cltv_sources_template.csv` with reliable public sources before implementing the collector.

Recommended source priority:

1. official bank pages and annual reports;
2. regulators and official public-sector sources;
3. consulting reports;
4. academic papers;
5. vendor technical explainers;
6. reputable business media.

Avoid using sources that are mostly SEO pages, thin marketing landing pages, or content without clear publication context.
